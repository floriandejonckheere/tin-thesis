    % Context (why is this work important)

    % Conclusion

    Ever since the dawn of dynamic web applications and social networks, there has been an increasing demand for performant storage systems to store the vast deluge of data that is generated by the social activity of users. This data is characterised by its size and inherently structured nature most types of social notification feeds can be reduced to a representation of an acyclic graph. In this paper, the current state of affairs pertaining to the storage of data in small to medium-size web applications will be summarized. An analysis of the structured data within the context of the Open Webslides project will be made, and an attempt will be made to match this analysis to a stable, scalable and maintained storage system. A proposed implementation for a normalized, efficient data schema will also be presented.
}
